1. Тесты для регистрации AuthenticationService
Успешная регистрация:
Данный тест проверяет успешную регистрацию нового пользователя.

Пользователь уже существует:
Данный тест проверяет случий, когда email уже зарегистрирован.

Пароли не совпадают:
Данный тест проверяет, когда при регистрации пароли не совпадают.

2. Тесты для аутентификация AuthenticationService
Успешная аутентификация:
Данный тест проверяет успешную аутентификацию.

Пользователь не найден:
Этот тест проверяет аутентификацию, когда пользователь не найден.
